## How to connect VScode to GitHub

### Step 1:

Download "Git"

### Step 2:

Open VScode

### Step 3:

Click "SHIFT + CTRL + P"

### Step 3:

Find "Git: Clone" & Click It

### Step 4:

Click "Clone from GitHub"

### Step 5:

Type in your GitHub Repository name

### Step 6:

Save to a place

### Step 7:

Open It
